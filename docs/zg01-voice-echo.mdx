先日 [ZG01](https://amzn.to/48YufOZ) を購入したのですが、PC から(正確には Yamaha ZG01 Voice)の音声にエコーがかかっているように聞こえていました。

EQ を OFF にしても変わらなくて困っていたのですが原因が分かったので書いておきます。

## 解決方法

Yamaha ZG Controller で GAME EFFECT を開きます。

そして USER PRESETS のどれかを選択して EDIT ボタンを押します。

![](https://gyazo.com/0c8c93437da62cad66d06a293fd3d388.png)

すると一番上に 3D CHAT SPACE という項目があると思うのですが、原因はこいつでした。

これを OFF にすればエコーが消えます。

![](https://gyazo.com/d799cf6f234edda5a4902268f4a4b1ab.png)

公式サイトによると、正確にはエコーではなくゲーム音とボイスチャットの音声を聞き分け出来るようにする機能のようです。

> 友達や仲間と同じ部屋でゲームをしているかのような音声体験ができます。相手の声を仮想立体空間へ配置することでゲーム音声との被りを避け、綺麗に聞き分けできるようにします。これにより快適なゲーム・ボイスチャットができます。

なので ON にしたプリセットも作っておいて、ボイスチャットをする場合としない場合(YouTube を見ながらゲームしたり)で切り替えると良いかもしれません。
