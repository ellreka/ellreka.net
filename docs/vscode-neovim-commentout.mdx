## 方法

`init.vim`に以下を追記する。

```vim
if exists('g:vscode')
  xmap gc  <Plug>VSCodeCommentary
  nmap gc  <Plug>VSCodeCommentary
  omap gc  <Plug>VSCodeCommentary
  nmap gcc <Plug>VSCodeCommentaryLine
endif
```

これで`react`とかで js 部分と jsx 部分でコメントアウトの種類を自動で切り替えてくれる。
